FAQ'S

What is this website for?

This website is built by sust cse 2018. Basically, this website is a part of an academic project/research work. This website will be used for collecting data, that will be used for the future research. And FYI, the research is about root/stem identification in bangla langugage.

How can I sign up?

Unfortunately, you can't sign up on your own. Usually, the admin assigns the selected users and only then you can get your username and password, which is required for signing in.

How does the webiste works ?

First, you'll have to log in using the assigned username and password. After that, you can access the data-entry page. In this page, you'll be given a word and you have to identify the root/stem of the word using the backspace key on your keyboard. Here you'll find three buttons : submit, skip and garbage. After you find the root, using backspace, press enter. The indentified root will be recorded on our database. If you are not sure about the root of a word, skip that word by pressing the skip button. If you think the given word is not a valid word, press the garbage button. Please make sure to log out before you exit the website.

Rules for Bangla Stemming

  • Stemming means stripping off the affixes(both suffix and prefix) to obtain the base form of the word, without changing the meaing of the given word. So, we need to strip the affixes to get the atomic stem, which is both meaningful and related to the given word. Example :

    সংবাদসমূহ -> সংবাদ; হরিণটার -> হরিণ; এঘটনা -> ঘটনা; তুলবেই -> তুলব (Here, if you strip the word to 'তুল' which means comparison, the meaing changes); ট্যাংকারডুবির -> ট্যাংকার; খ্রিস্টধর্মাবলম্বী -> খ্রিস্ট; ক্রমাঙ্কিত -> ক্রম; বংশধরদেরও -> বংশ; কার্যপরিচালনার -> পরিচালনা; সুবাসনা -> বাসনা

  • Please be careful about overstemming. Overstemming changes the meaing from the original word. Example :

    যন্ত্রণা -> যন্ত্রণা (not যন্ত্র) ; খালামনি -> খালামনি (not খাল); মাটি -> মাটি (not মা); নিবন্ধন -> নিবন্ধন (not বন্ধন)

  • Don't modify the name of the places, people or anything specific. Example :

    ইন্ডিয়াটাইম্‌স -> ইন্ডিয়াটাইম্‌স; সামীউল -> সামীউল (Don't make it সামী); আলকনাথ -> আলকনাথ

  • For words consist of two stems, store the first one as the stem. In this case, please be careful that the contextual meaning of the word remains the same. For some cases, you may need to select the second stem. If you have confusion about which one to chose as the stem, it is advised to skip the word or chose the first one.Example :

    স্মৃতিকথা -> স্মৃতি; বইপুস্তক -> বই; উদ্দেশ্যসাধন -> উদ্দেশ্য; নগরভবন -> ভবন (Here the main context of the word is a 'building', rather than a 'city'); মাতৃভাষা -> ভাষা; ছায়াছবি -> ছবি

  • Stop stemming if the spelling of the word becomes wrong. Example :

    আনুষ্ঠানিকতাটুকু -> আনুষ্ঠানিক (not আনুষ্ঠান as the spelling is wrong); সামাজিকতা -> সামাজিক (not সামাজ); দালিলিকভাবে -> দালিলিক (not দালিল)

  • Don't strip the affixes if the meaing changes fully. Example :

    একঠোঁটা -> একঠোঁটা (Don't make is এক or ঠোঁট, as the meaning changes completely)

  • Please refrain yourself from submiting stems, you are not sure about. We highly recommend you to skip those words. If you have any confusion, feel free to ask the admins.
